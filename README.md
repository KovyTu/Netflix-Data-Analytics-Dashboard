# Netflix Content Analytics Dashboard

## Project Overview

This project is an interactive Power BI dashboard that analyzes 8,809 Netflix titles to identify trends in content type, ratings, genres, and release years. The dashboard provides visual insights into Netflix's content library and demonstrates data visualization and business intelligence techniques using Power BI.

## Objectives

- Analyze the distribution of Movies and TV Shows on Netflix.
- Identify the most common content ratings.
- Examine content growth over time.
- Explore the most popular genres available on Netflix.
- Present key metrics through an executive-style dashboard.

## Tools Used

- Power BI
- Data Visualization
- Business Intelligence Reporting
- Data Analysis

## Dashboard Features

### KPI Cards
- Total Titles
- Total Movies
- Total TV Shows

### Visualizations
- Content Type Distribution
- Top Content Ratings
- Netflix Content Released by Year
- Ratings Distribution
- Top Netflix Genres
- Countries with Most Netflix Content

## Key Insights

- Movies account for approximately 70% of Netflix's content library.
- TV Shows account for approximately 30% of available titles.
- TV-MA is the most common content rating.
- Netflix experienced significant content growth after 2015.
- Drama and Documentary categories are among the most common genres.
